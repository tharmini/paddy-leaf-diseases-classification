# Haar Wavelet-Guided Dynamic Feature Pyramid Network for Efficient Paddy Leaf Disease Classification

This repository presents a novel deep learning framework integrating Haar Wavelet decomposition with a Dynamic Feature Pyramid Network (DFPN) to enhance multi-scale feature representation for accurate and computationally efficient paddy leaf disease classification.

---

##  Overview

Paddy leaf diseases significantly impact crop yield and food security. Traditional CNN-based classification models often struggle with fine-grained texture variations and multi-scale disease patterns.

To address these challenges, we propose:

- ✔ Haar Wavelet-guided feature enhancement  
- ✔ Dynamic Feature Pyramid Network for multi-scale fusion  
- ✔ Lightweight and computationally efficient design  
- ✔ Improved classification performance on paddy leaf disease datasets
